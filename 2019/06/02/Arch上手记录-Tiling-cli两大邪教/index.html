科普

什么是窗口管理器

一个桌面环境通常包含几大核心组件，窗口管理器便是其中之一，主要负责用户对桌面窗口的交互行为。通常包含两种类型的窗口管理：

  1. Stacking

    堆叠式窗口，最为普遍的类型，一般以鼠标操作为主，键盘辅助，可以任意拖动浮动窗口，窗口与窗口之间能够互相堆加在一起。

    Win，Mac以及大部分主流Linux桌面都采用堆叠窗口。

  2. Tiling

    平铺式窗口，以键盘为操作核心的形式，抛弃了大部分鼠标操作，利用快捷键创建新窗口，同时管理器会自动调整新窗口位置，通过平铺形式提升界面的利用率。上手难度相对更大，但在定制化和效率上更佳。

  3. Dynamic

    动态式ss窗口

什么是CLI

Command Line Interface(命令行界面)，抛弃图形界面，以终端交互的形式进行运作的软件，通常都拥有以下几种属性：

  1. 轻量级
  2. 高效率，主打快捷键
  3. 界面简洁透明
  4. 可扩展性强
  5. 不同CLI之间可以配合

同时上手成本较高也是一个共同点，学会查阅文档是使用CLI应用的第一前提。

专注于终端键盘操作这一性质完美契合了Tiling的逻辑，两者结合可以彻底放弃对鼠标的依赖。

I3

如果希望体验Tiling的高效简洁，i3是非常不错的选择。相比同类软件，i3在功能上相对偏少，不过胜在配置简单易懂，官网附带详细的说明文档，非常适合新人上手。
